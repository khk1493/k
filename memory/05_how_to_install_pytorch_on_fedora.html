Hot to install PyTorch on linux(fedora)



날짜: 2018-09-27


  1. Anaconda3 설치
  2. Pycharm-community 설치
  3. 아나콘다 프롬프트 창을 연다.

  4. 가상환경 생성 export PATH=~/anaconda2/bin:$PATH

    # conda create --name lito4 python=3.6

  5. 가상환경 안에 들어간다.

    # source activate lito4

  6. pip update

    (lito4) # ? pip

  7. conda upgrade

    (lito4) # conda update conda

  8. pytorch 설치

    (lito4) # conda install pytorch-cpu torchvision-cpu -c pytorch

  9. 가상환경 위치

    root/anaconda2/envs/lito4/bin/python3.6

  10. vi 편집

    vi 파일이름, i 입력모드, Esc, :wq

  11. 파이참 사용

    log-in 모드에서 ./lito4.sh 입력하고 기다린다.

  12. 가상환경 이름, 프로젝트 이름

    모두 lito4
