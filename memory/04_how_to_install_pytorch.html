Hot to install PyTorch on windows



날짜: 2018-09-27


  1. Anaconda3 설치
  2. Pycharm-community 설치
  3. 아나콘다 프롬프트 창을 연다.

  4. 가상환경 생성

    (base) C:\Users\user> conda create --name (가상환경 이름) python=3.6

  5. 가상환경 안에 들어간다.

    (base) C:\Users\user> activate (가상환경 이름)

  6. pip update

    (가상환경 이름) C:\Users\user> python -m pip install --upgrade pip

  7. conda upgrade

    (가상환경 이름) C:\Users\user> conda upgrade conda

  8. pytorch 설치

    (가상환경 이름) C:\Users\user> conda install pytorch-cpu -c pytorch

  9. torchvision 설치

    (가상환경 이름) C:\Users\user> pip3 install torchvision
