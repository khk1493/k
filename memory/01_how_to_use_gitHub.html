깃헙 윕 호스팅으로 사용하기

  1. 깃헙에 가입한다.
  2. new repository 버튼이나 + 표시를 클릭하여 나오는 new repository를 클릭한다.
  3. 저장소 이름을 입력한다.
  4. 아래에 "initialize this repository with a README"를 체크한다.
  5. create repository를 클릭한다.
  6. 우측상단의 "upload files" 버튼을 클릭한다.
  7. upload할 파일을 드래그한다.
  8. 업로드한 파일의 내용을 기재한다.
  9. 아래 연두색 버튼의 commit changes를 클릭한다.
  10. 업로드한 파일 리스트를 확인한다.
  11. 우측 상단의 "settings" 메뉴를 클릭한다.
  12. 중간쯤 githun pages 아래 none을 클릭하여 "master branches"를 선택하고 save한다.
  13. gitHub pages 아래 있는 주소를 클릭한다.
